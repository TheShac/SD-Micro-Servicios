# SD Micro Frontend (React + Vite + Tailwind)

## Quick start
1. Copy `.env.example` to `.env` and adjust service URLs if needed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:5173

This frontend provides pages: Login, Register, Recovery, Verification and Dashboard.
